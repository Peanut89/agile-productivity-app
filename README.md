# Agile Productivity App

A personal productivity / checklist app built around the agile process — **Epics → Stories → Tasks**, organized into **Sprints** with a backlog and a board.

Local-first, designed as a **PWA** (installable on mobile via "Add to Home Screen"), with a clean path to wrap as a native app via Capacitor later.

## Status

🚧 Early scaffolding.

## Planned stack

- React + TypeScript + Vite
- vite-plugin-pwa (offline + installable)
- IndexedDB via Dexie (local-first storage)
- Tailwind CSS
- Zustand (state)
